## ğŸ“¦ Installation

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Frontend Setup
```bash
# Clone the repository
git clone https://github.com/jjulito/spotify-clon.git
cd spotify-clon

# Install dependencies
npm install

# Start development server
npm start
```

### Backend Setup
```bash
# Navigate to backend directory
cd backend

# Install dependencies
npm install

# Start development server
npm run dev
```
